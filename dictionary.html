The recommended core data set in the WHO Digital Documentation of COVID-19 Certificates guidance is intended to include the critical data required for interoperability, specific to the scenarios of use defined and driven by the public health needs. A published version of the data dictionary is available in spreadsheet format in Web Annex A of the DDCC:Vaccination Status guidance and in an annex to the DDCC:Test Result guidance. Logical Models DDCC:Vaccination Status: Continuity of Care Core Data Set DDCC:Vaccination Status: Proof of Vaccination Core Data Set DDCC:Test Result Core Data Set, which includes Proof of Negative SARS-CoV-2 Test Result and Proof of Previous SARS-CoV-2 Infection.
